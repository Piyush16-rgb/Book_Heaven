# Book Heaven

**Book Heaven** is a React Native application designed as an online e-commerce platform for buying books. This app provides users with a seamless browsing experience, allowing them to explore books, view details, and make purchases.

## Overview

Book Heaven is a mobile application built with React Native that offers a rich user experience for book enthusiasts. Users can browse various book categories, search for specific titles, view detailed information about each book, and add items to their cart for purchase.

## Features

- **Browse Books**: View books organized by categories.
- **Search Functionality**: Quickly find books using the search bar.
- **Book Details**: Access detailed information, including descriptions and prices.
- **Shopping Cart**: Add books to the cart and view the cart summary.
- **User Authentication**: Sign up and log in to manage your profile and orders.

- ## Usage

- **Explore the App**: Open the app and navigate through different sections to browse books, view details, and add items to your cart.
- **Search for Books**: Use the search bar to find specific titles or authors.
- **Manage Cart**: Add items to your cart and proceed to checkout when ready.
- **Sign In/Sign Up**: Access authentication features to manage your account.
